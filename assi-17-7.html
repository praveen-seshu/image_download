Text elements

In HTML (HyperText Markup Language), text elements are used to structure and format textual content on a webpage. These elements help browsers understand how different parts of the text should appear and behave.
  1. Headings 'h1' to 'h6' Used to define headings of different levels. 'h1' is the highest (most important), and 'h6' is the lowest.
  2. Paragraph 'p' Used to define a paragraph of text.
  3. Bold ('b' or 'strong') 'b' makes text bold without implying importance.
    4. 'strong' also makes text bold, but emphasizes importance semantically.
  4. 'h1' to 'h6' – Headings Define headings from largest ('h1') to smallest ('h6').
  5. 'p' – Paragraph Defines a block of text as a paragraph.
    6. l
  6. 'b' – Bold Displays text in bold without semantic importance.
  7. 'strong' – Strong Importance Displays text in bold and indicates strong importance.
  8. 'i' – Italic Displays text in italic for stylistic purposes.
  9. 'em' – Emphasis Displays text in italic and adds semantic emphasis.
  10. 'u'Underline Underlines the text.
  11. 'br' Line Break Inserts a line break within text.
  12. 'hr' – Horizontal Rule Inserts a horizontal line to separate content.
  13. 'span' – Inline Container Used to apply styles to inline text.
  14. 'pre' – Preformatted Text Preserves spaces and line breaks in the text.
  15. 'blockquote' – Block Quotation Indicates a large quoted section from another source.
  16. 'code' – Code Text Displays text as programming code.
  17. 'sup' – Superscript Displays text slightly above the normal line.
  18. 'sub' – Subscript Displays text slightly below the normal line.

Images

The 'img' tag in HTML is used to embed an image on a webpage. It is an empty (self-closing) tag and requires the src attribute to specify the image source and the alt attribute to provide alternative text if the image cannot be displayed.

Anchor tags

The 'a' tag in HTML is used to create hyperlinks. It links one page to another using the href attribute, which specifies the URL or location to navigate to when the link is clicked.
